# FinalProject

Spring MVC 모듈을 사용하여 그룹웨어 사이트를 제작한 팀프로젝트입니다.

# 개발환경

● OS: Windows 11

● Language: Java 1.8.0

● DBMS: Oracle 18c

● FrameWork: Spring 5.3.22

● Server: Tomcat 9.0

● Front-End: HTML, CSS, javascript, jQuery

# 제작기간

● 2023.12.01 ~ 2024.01.12

# 팀원

● 박윤수 : 일정관리, 자원예약, 증명서 및 문서발급, 웹 채팅

● 박승우 : 사내메일,가입심사

● 김진영 : 전자결제, 결제관리

● 배민준 : 휴관관리, 근무관리

● 이요섭 : 로그인, index, 인사관리

● 김경민 : 회원가입, 급여관리, 조직도, 게시판
